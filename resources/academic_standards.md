# Академические стандарты и чеклисты

---

## Чеклист готовности дипломной / магистерской работы

### Структура и содержание
- [ ] Исследовательский вопрос сформулирован чётко и присутствует в тексте
- [ ] Актуальность темы обоснована конкретно (не "тема важна, потому что важна")
- [ ] Цели и задачи соответствуют друг другу и достигнуты в работе
- [ ] Объект и предмет исследования определены и различимы
- [ ] Теоретическая база чётко обозначена и обоснована
- [ ] Методология описана и применена последовательно
- [ ] Каждая глава выполняет свою задачу в рамках общей логики
- [ ] Выводы каждой главы связаны с общими выводами
- [ ] Общие выводы отвечают на исследовательский вопрос
- [ ] Заключение не вводит новых данных или аргументов

### Работа с источниками
- [ ] Количество источников соответствует требованиям
- [ ] Источники актуальны (не более 5-7 лет, если не классика)
- [ ] Есть источники на иностранных языках (если требуется)
- [ ] Все источники из списка литературы упомянуты в тексте
- [ ] Все упоминания в тексте есть в списке литературы
- [ ] Прямые цитаты оформлены корректно (кавычки + ссылка со страницей)
- [ ] Парафраз атрибутирован (ссылка без кавычек)
- [ ] Нет самоцитирования из неопубликованных работ без оговорки

### Оформление (ГОСТ)
- [ ] Титульный лист по образцу учебного заведения
- [ ] Содержание (оглавление) с номерами страниц
- [ ] Нумерация страниц (обычно с 3-й страницы, кроме титула)
- [ ] Список литературы оформлен по ГОСТ Р 7.0.100-2018
- [ ] Приложения пронумерованы и упомянуты в тексте
- [ ] Таблицы и рисунки пронумерованы, подписаны, упомянуты в тексте
- [ ] Отступы, межстрочный интервал, кегль соответствуют требованиям

### Качество текста
- [ ] Нет плагиата (проверено в системе антиплагиата)
- [ ] Нет незакрытых флагов [VERIFY], [SOURCE NEEDED] и т.п.
- [ ] Проведена финальная вычитка (желательно вслух)
- [ ] Нет грамматических и орфографических ошибок
- [ ] Терминология последовательна (один термин — одно понятие)

## Чеклист академической статьи

- [ ] Title отражает суть исследования (не слишком широкий, не слишком узкий)
- [ ] Abstract: актуальность, цель, методы, результаты, выводы — все 5 элементов
- [ ] Keywords: 5-8 слов, не повторяют заглавие буквально
- [ ] Введение: gap в литературе обозначен явно
- [ ] Обзор литературы: аналитический, не реферативный
- [ ] Методы: воспроизводимы (другой исследователь может повторить)
- [ ] Результаты отделены от интерпретации
- [ ] Обсуждение: связывает результаты с литературой и тезисом
- [ ] Ограничения исследования названы
- [ ] Вклад работы сформулирован явно
- [ ] References в нужном формате (проверены)
- [ ] Объём соответствует требованиям журнала

## Основные требования ГОСТ Р 7.0.11 (диссертации)

### Оформление текста
- Поля: левое — 30 мм, правое — 10 мм, верхнее — 20 мм, нижнее — 20 мм
- Шрифт: Times New Roman, кегль 14 (или Arial 12)
- Межстрочный интервал: 1,5
- Абзацный отступ: 12,5 мм (1,25 см)
- Выравнивание основного текста: по ширине

### Нумерация
- Страницы: арабские цифры, центр нижнего колонтитула
- Титульный лист и оглавление включаются в общую нумерацию, но номер не ставится
- Разделы: арабские цифры (1, 2, 3)
- Подразделы: 1.1, 1.2, 2.1 и т.д.
- Точка после номера раздела в тексте — ставится; в оглавлении — по образцу

### Список литературы
- ГОСТ Р 7.0.100-2018 "Библиографическая запись. Библиографическое описание"
- Оформление: авторы (Фамилия И.О.), через пробел — Название. — Место: Издательство, Год. — Страниц с.
- Статьи: автор // Название журнала. — Год. — Т. X, № X. — С. XX–XX.
- Иностранные источники — в конце списка или в общем алфавите (по требованию)

## Топ-15 типичных ошибок в академическом тексте

1. **Декларативное введение без gap** — "Тема актуальна" без доказательства что именно не изучено
2. **Цель ≠ задачам** — задачи не ведут к цели или выходят за её рамки
3. **Плавающие цитаты** — цитата без введения: кто сказал, в каком контексте
4. **Выводы-пересказы** — вывод повторяет сказанное выше вместо того чтобы делать умозаключение
5. **Источники как украшение** — ссылка в конце абзаца, который ни на что не опирается по сути
6. **Терминологическая непоследовательность** — один объект называется тремя словами
7. **Перегруженные абзацы** — 5+ идей в одном абзаце без структуры
8. **Методология без применения** — метод описан, но не виден в анализе
9. **Избыточные вводные** — каждое второе предложение начинается с "Следует отметить, что..."
10. **Потеря исследовательского вопроса** — к середине работы ИВ забыт
11. **Обзор литературы как аннотированный список** — "Иванов пишет... Петров пишет..." без синтеза
12. **Самоочевидные утверждения** — факты, которые читатель и так знает, поданные как открытие
13. **Несогласованные выводы** — заключение приходит к выводам, не подготовленным текстом
14. **Неоформленный парафраз** — чужая идея без ссылки, потому что "это общеизвестно"
15. **Неконкретные прилагательные** — "значительный", "существенный", "важный" без данных

## Принципы академической честности

- **Атрибуция:** любая заимствованная идея — прямая цитата или парафраз — требует ссылки
- **Прямое цитирование:** только когда точная формулировка важна; остальное — парафраз
- **Самоплагиат:** повторное использование собственных опубликованных работ требует оговорки
- **Источники из вторых рук:** не цитировать источник, который не читал; использовать форму "цит. по:"
- **ИИ-генерация:** уточни требования своего университета; если запрещено — не использовать
- **Данные:** не подгонять данные под гипотезу; если гипотеза не подтвердилась — это тоже результат
- **Рецензия:** получение неопубликованных рукописей на рецензию — конфиденциальная информация
