# Форматы цитирования

---

## ГОСТ Р 7.0.100-2018 (Россия)

### Книга (монография)
```
Фамилия И. О. Название монографии / И. О. Фамилия. — Место издания : Издательство, Год. — Количество с.
```
Пример:
```
Лотман Ю. М. Семиосфера / Ю. М. Лотман. — СПб. : Искусство-СПБ, 2000. — 704 с.
```

### Книга с несколькими авторами
```
Фамилия И. О. Название / И. О. Фамилия, И. О. Фамилия2. — Место : Издательство, Год. — Страниц с.
```

### Статья в журнале
```
Фамилия И. О. Название статьи / И. О. Фамилия // Название журнала. — Год. — Т. X, № X. — С. XX–XX.
```

### Глава в сборнике / коллективной монографии
```
Фамилия И. О. Название главы / И. О. Фамилия // Название сборника / под ред. И. О. Редактор. — Место : Изд-во, Год. — С. XX–XX.
```

### Диссертация
```
Фамилия И. О. Название : дис. ... канд. (д-ра) [наук, специальность] / И. О. Фамилия. — Место, Год. — Страниц с.
```

### Интернет-ресурс
```
Фамилия И. О. Название [Электронный ресурс] / И. О. Фамилия // Название сайта. — URL: https://... (дата обращения: ДД.ММ.ГГГГ).
```

### Ссылки в тексте (ГОСТ)
- Концевые сноски: ¹ или [1]
- Внутритекстовые: [Фамилия, Год, с. X] — допускается
- Подстрочные: по решению учебного заведения

## APA 7 (American Psychological Association)

### Книга
```
Фамилия, И. О. (Год). Название книги (курсив). Издательство.
```
Пример:
```
Goffman, E. (1959). The presentation of self in everyday life. Doubleday.
```

### Статья в журнале
```
Фамилия, И. О., & Фамилия2, И. О. (Год). Название статьи. Название Журнала (курсив), Том(Номер), Страницы. https://doi.org/...
```

### Глава в редактированной книге
```
Фамилия, И. О. (Год). Название главы. В И. О. Редактор (Ред.), Название книги (курсив) (стр. XX–XX). Издательство.
```

### Ссылки в тексте (APA)
- Одиночный автор: (Фамилия, Год) или Фамилия (Год)
- Два автора: (Фамилия & Фамилия, Год) — всегда
- Три и более: (Первый et al., Год)
- Прямая цитата: (Фамилия, Год, с. XX) или (Фамилия, Год, p. XX)
- Несколько источников: (Автор1, Год; Автор2, Год) — алфавитный порядок

## Chicago (Notes-Bibliography, 17th ed.)

### Книга — сноска
```
Имя Фамилия, Название книги (Место: Издательство, Год), страница.
```

### Книга — библиография
```
Фамилия, Имя. Название книги. Место: Издательство, Год.
```

### Статья — сноска
```
Имя Фамилия, "Название статьи," Название журнала Том, № Номер (Год): Страница.
```

### Ссылки в тексте
Chicago использует сноски (footnotes). При первом упоминании — полное описание, при повторном: Фамилия, Краткое название, страница. или Ibid., страница.

## MLA 9 (Modern Language Association)

### Книга
```
Фамилия, Имя. Название книги. Издательство, Год.
```

### Статья
```
Фамилия, Имя. "Название статьи." Название журнала, т. X, № X, Год, с. XX–XX.
```

### Ссылки в тексте (MLA)
- Внутритекстовые: (Фамилия страница) — без запятой, без "с."
- Пример: (Goffman 45) или Goffman argues... (45)
- Если несколько работ одного автора: (Фамилия, Краткое название страница)

---

## Быстрое сравнение форматов

| Элемент | ГОСТ | APA | Chicago | MLA |
|---------|------|-----|---------|-----|
| Ссылка в тексте | [1] или сноска | (Автор, Год) | Сноска | (Автор стр.) |
| Год в библиографии | В конце | После автора | В конце | В конце |
| Имена авторов | Фамилия И.О. | Фамилия, И. | Имя Фамилия | Фамилия, Имя |
| Курсив | Не всегда | Название книги/журнала | Название книги/журнала | Название книги/журнала |

## Частые ошибки при цитировании

1. **Цитата без страницы** — при прямом цитировании всегда нужна страница
2. **Парафраз с кавычками** — или цитируй точно (с кавычками и страницей), или парафразируй без кавычек
3. **Ibid. / Там же** — только если источник тот же, что в непосредственно предшествующей сноске
4. **Дата обращения к URL** — обязательна; используй реальную дату когда проверял ссылку
5. **"Цитата цитаты"** — оформляй как "цит. по:" / "qtd. in:" с указанием источника, в котором нашёл
6. **Неполное описание** — нет издательства, нет года, нет тома/номера журнала — всё это нужно
