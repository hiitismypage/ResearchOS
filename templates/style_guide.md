## Общий тон

**Тон текста:**
- [ ] Строго академический (дистанцированный, безличный)
- [ ] Академический с авторским присутствием
- [ ] Полуформальный (допустим более живой язык)
- Описание: [ЗАПОЛНИ своими словами]

**Дистанция с читателем:**
[ЗАПОЛНИ: "автор обращается к читателю как к равному коллеге" / "нейтральный научный регистр без обращений" / другое]

---

## Залог и лицо

**Пассивный залог:**
- [ ] Часто (основной инструмент)
- [ ] Иногда (там где нет логичного подлежащего)
- [ ] Редко (только в методологическом разделе)

**Первое лицо:**
- [ ] Допустимо: "я полагаю", "я выделяю"
- [ ] Только: "автор настоящей работы"
- [ ] Только: "мы рассматриваем" (авторское "мы")
- [ ] Запрещено — только безличные конструкции

## Синтаксис

**Длина предложений:**
[ЗАПОЛНИ: "преимущественно средние (20-35 слов), сложные конструкции допустимы в теоретических разделах" / другое]

**Абзацы:**
- [ ] Правило одной мысли (каждый абзац — один тезис)
- [ ] Допустимы абзацы с несколькими связанными мыслями
- Минимальная длина абзаца: [ЗАПОЛНИ: "3 предложения"]
- Максимальная длина абзаца: [ЗАПОЛНИ: "150-200 слов"]

**Вводные конструкции:**
- [ ] Минимум (только по необходимости)
- [ ] Допустимы для логических переходов
- [ ] Разрешены широко

## Цитирование и ссылки

**Формат ссылок в тексте:**
- [ ] (Автор, Год: страница) — APA
- [ ] [Номер] — нумерованный список
- [ ] Сноски (ГОСТ)
- [ ] Другой: [ЗАПОЛНИ]

**Введение цитат:**
[ЗАПОЛНИ предпочтительные конструкции: "По мнению X (Год: с.)..." / "Как отмечает X, '...' (Год: с.)" / другое]

**Парафраз vs прямое цитирование:**
[ЗАПОЛНИ: "предпочитать парафраз, прямые цитаты только для ключевых определений и полемики" / другое]
