---
name: find-quote
description: Ищет релевантные цитаты и тезисы по теме из обработанных карточек sources/notes/
model: sonnet
---

Аргумент: тема или тезис, для которого нужны цитаты или обоснование. Например:
- `/find-quote гендерный контракт в советском обществе`
- `/find-quote репрезентация как конструирование реальности`

## Шаги

1. **Прочитай все файлы** из `projects/<проект>/sources/notes/`
   (кроме `_template.md`).
   ВАЖНО: читай ТОЛЬКО карточки (notes/), не оригинальные файлы из files/ —
   это экономит токены.

2. **Найди релевантные источники** по:
   - Тегам
   - Разделу "Тезис работы" и "Ключевые аргументы"
   - Разделу "Связь с моей работой"
   - Разделу "Релевантные цитаты"

3. **Выведи структурированный список:**

```
## Результаты поиска по теме: "<запрос>"

### Релевантные источники

**[Автор, Год] Краткое название** → notes/<файл>.md
Тезис: ...
Цитата: "..." (с. XX)
Как использовать: поддерживает / опровергает / иллюстрирует
Предлагаемый ввод в текст: "По мнению Автора (Год), «...»"

---

**[Автор2, Год]...**
...

### Источники для полемики (опровергают)
...

### Не найдено в notes/
Для следующих аспектов нужен дополнительный поиск:
- ...
```

4. **Если нужных цитат нет в notes/** — скажи явно, предложи запустить `/find-sources`
   для поиска новых источников. Не выдумывай.
