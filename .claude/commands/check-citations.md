Проверяет все незакрытые флаги в тексте и предлагает решения из обработанных источников.

## Шаги

1. **Найди все файлы** с флагами в `projects/<проект>/chapters/`:
   Ищи строки содержащие `[SOURCE NEEDED]` и `[VERIFY]`.

2. **Для каждого флага:**
   - Покажи контекст (предложение с маркером)
   - Поищи в `sources/notes/` подходящий источник
   - Предложи конкретную цитату ИЛИ скажи "нет в notes/ — нужен ручной поиск"

3. **Выведи таблицу:**

```
| # | Флаг | Файл | Контекст | Статус | Предложение |
|---|------|------|----------|--------|-------------|
| 1 | [SOURCE NEEDED] | 02_teoria.md | "...конструирует смыслы..." | ✅ Есть в notes/ | Hall (1997), с. 25 — notes/hall_1997.md |
| 2 | [VERIFY] | 01_vvedenie.md | "...принято в 1968 году..." | ❌ Нет в notes/ | Нужен ручной поиск |
| 3 | [SOURCE NEEDED] | 03_analiz.md | "...двойная нагрузка..." | ✅ Есть в notes/ | Темкина (2007) — notes/temkina_2007.md |
```

4. **Итог:**
```
Итого флагов: X
✅ Закрыто из notes/: X
❌ Требует ручного поиска: X

Рекомендация: для незакрытых — запустить /find-sources "<тема>"
```

ВАЖНО: читай только `notes/`, не оригинальные файлы из `files/`.
