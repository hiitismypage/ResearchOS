Ты анализируешь логическую и аргументационную структуру текста или тезиса.
Строишь карту аргументации и выявляешь слабые места.

## Аргумент скилла

Раздел или конкретный тезис для анализа. Например:
- `/argument-check chapters/03_analysis.md`
- `/argument-check "Цифровизация образования приводит к снижению критического мышления"`

## Что прочитать перед анализом

- Указанный файл или тезис
- `projects/<проект>/CONTEXT.md` — исследовательский вопрос (относительно него проверяем)

## Структура анализа

### 1. Центральный тезис

Сформулируй центральный тезис раздела (как ты его понял):
> "..."

Вопросы:
- [ ] Тезис явно сформулирован в тексте?
- [ ] Тезис достаточно конкретен (не слишком широкий)?
- [ ] Тезис отвечает на исследовательский вопрос?

### 2. Карта аргументации

Визуальное дерево в markdown:

```
ТЕЗИС: "..."
├── Аргумент 1: "..."
│   ├── Доказательство: "..." [источник / данные]
│   └── [WEAK ARG] — если слабо подкреплён
├── Аргумент 2: "..."
│   ├── Доказательство: "..."
│   └── Контраргумент: "..." → Опровержение: "..." / [НЕ РАССМОТРЕН]
└── Аргумент 3: "..."
    └── ...
```

### 3. Проверка логических ошибок

Проверь наличие следующих fallacies (только те, что найдёшь в тексте):

| Ошибка | Описание | Пример из текста |
|--------|----------|-----------------|
| Поспешное обобщение | Вывод из слишком малого числа примеров | |
| Ложная дихотомия | "Либо А, либо Б" когда есть другие варианты | |
| Circulus in probando | Аргумент предполагает то, что доказывает | |
| Ad populum | "Все так считают, значит это верно" | |
| Post hoc | А случилось до Б, значит А — причина Б | |
| Соломенное чучело | Опровержение упрощённой версии аргумента оппонента | |
| Ложная аналогия | Сравнение несопоставимых вещей | |
| Апелляция к авторитету | Авторитет вместо аргумента | |

### 4. Контраргументы

- [ ] Опровержение убедительно?
- Неучтённые контраргументы (которые стоит рассмотреть): [список]

### 5. Достаточность доказательств

- [ ] Каждый ключевой аргумент подкреплён данными/ссылками?
- [ ] Соотношение утверждений и доказательств сбалансировано?
