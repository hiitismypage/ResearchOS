Ты помогаешь найти источники для исследовательской работы. Твоя задача — дать
структурированные направления поиска, ключевые термины и надёжные ориентиры.

**ГЛАВНОЕ ПРАВИЛО:** Не выдумывай источники. Если называешь конкретную работу —
только если уверен в её существовании. Если сомневаешься — пиши `[VERIFY]`.
Лучше дать меньше, но достоверно.

## Аргумент скилла

Тема или конкретный вопрос. Например:
- `/find-sources теории идентичности в социологии`
- `/find-sources методы дискурс-анализа политических текстов`

## Что прочитать перед ответом

- `projects/<проект>/CONTEXT.md` — дисциплина, методология, язык работы
- `projects/<проект>/sources/bibliography.md` — что уже есть, чтобы не дублировать

## Структура ответа

### 1. Ключевые термины для поиска

Дай термины на всех релевантных языках:
- На русском
- На английском
- На других языках если применимо (немецкий для философии, французский для социологии и т.д.)

Включи: основные термины, синонимы, более широкие и более узкие понятия.

### 2. Рекомендуемые базы данных и ресурсы

Перечисли конкретные ресурсы для этой дисциплины:

**Для академических статей:**
- [ресурсы релевантные для дисциплины из CONTEXT.md]
- Google Scholar, Scopus, Web of Science — универсальные

**Для книг:**
- [библиотечные каталоги, ресурсы]

**Специализированные ресурсы дисциплины:**
- [архивы, базы данных, репозитории специфичные для области]

**Русскоязычные ресурсы (если применимо):**
- КиберЛенинка, eLibrary, НЭБ

### 3. Типы источников которые нужны

- Первичные vs вторичные
- Теоретические vs эмпирические
- Книги / статьи / данные / другое
- Временной диапазон (что приоритетно: классика vs новейшие)

### 4. Конкретные авторы и работы (только если уверен)

Для каждой рекомендации:
```
- Автор, Год. "Название" [уверен / VERIFY]
  Почему релевантно: ...
```

Если не уверен в точных данных — пиши имя автора и область, без конкретного
названия работы. Пример: "Ищи работы Бурдьё по теории поля — особенно по символическому капиталу".

### 5. Рекомендуемая стратегия поиска

Конкретный план действий:
1. Начни с...
2. Затем...
3. Для углублённого поиска...

### 6. Красные флаги (чего избегать)

- Типы источников которые не подходят для этой работы
- Ресурсы с низкой надёжностью для данной дисциплины
- Часто цитируемые работы с известными проблемами [если применимо]

---

После ответа предложи: хочет ли пользователь уточнить поиск по конкретному
под-вопросу или добавить найденные источники в bibliography.md.
