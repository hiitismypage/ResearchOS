## Тезис работы (1-2 предложения)

Стандартный учебник по статистическому анализу категориальных данных. Охватывает распределения для категориальных переменных, методы для таблиц сопряжённости, логистическую регрессию, логлинейные модели, анализ соответствий и методы для кластеризованных данных.

---

## Ключевые аргументы / методы

1. **Глава 2 (Describing Contingency Tables):** Вероятностная структура таблиц сопряжённости, сравнение пропорций, частичные ассоциации.

2. **Глава 3 (Inference for Contingency Tables):**
   - §3.2: Тест χ² на независимость в двухмерных таблицах сопряжённости
   - §3.3: Методы post-hoc после χ²-тестов (стандартизированные остатки)
   - §3.5: Тесты для малых выборок — точный тест Фишера (Fisher's exact test), критерии независимости при малых N

3. **Глава 9.6 (Association Models, Correlation Models, and Correspondence Analysis):** Анализ соответствий как метод визуализации ассоциаций в таблицах сопряжённости.

4. **Глава 10.5 (Measuring Agreement Between Observers):** Методы измерения согласия между наблюдателями для категориальных данных.

---

## Методология

Учебник уровня graduate. Охватывает как классические методы (χ², Fisher), так и современные подходы (GLM, смешанные модели). Глава 16 — исторический обзор метода.

---

## Релевантные цитаты

Учебник, прямое цитирование нецелесообразно. Используется как методологический авторитет для chi-square, Fisher's exact и Cramér's V.

---

## Связь с моей работой

- **Раздел 3.5 (Statistical Analysis):** Методологическое обоснование использования χ²-теста для проверки независимости между переменными (тип легитимизации × геополитический блок); Fisher's exact test для малых N при анализе подгрупп; Cramér's V как мера силы ассоциации.
- Стандартная ссылка при описании chi-square и Fisher's exact в академических работах по политическим наукам.

---

## Критика / слабые места

- Учебник ориентирован на статистику и биостатистику; примеры не из социальных наук. Но методологические принципы универсальны.
- 2nd ed. (2002) — актуальная версия для классических методов; более новые подходы описаны в 3rd ed. (2013). Для χ², Fisher's exact, Cramér's V разница несущественна.
