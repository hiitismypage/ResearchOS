# 4 Математическая модель экономики с 5 отраслями

Предлагаемая модель имеет двухуровневую структуру. На макроуровне сохраняется динамический межотраслевой баланс, обеспечивающий согласованность производственных, инвестиционных и конечных потоков. На микроуровне каждая отрасль представляется совокупностью фирм, различающихся по капиталу, технологическому уровню, ожидаемому спросу и инвестиционной активности. В результате агрегированные матрицы $A(t)$ и $B(t)$ становятся функциями состояний и решений агентов, а макроэкономическая динамика интерпретируется как результат децентрализованного взаимодействия фирм \[8; 12\].

## 4.1 Отраслевой состав модели и выбор числа агентов

Для прикладной реализации модели выделяется пять отраслей, образующих минимально достаточный производственный контур экономики:

$$J = \left\{ 1,2,3,4,5 \right\}.\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad\quad(4.1)$$

Пусть отрасли имеют следующую структуру:

- Добывающая промышленность, $N_{1} = 32$. Отрасль поставляет нефть, газ, уголь и иные первичные ресурсы для всех остальных отраслей;

- Электроэнергетика, $N_{2} = 27$. Отрасль формирует универсальный промежуточный ресурс, используемый всеми секторами;

- Обрабатывающая промышленность, $N_{3} = 41$. Отрасль производит промежуточные товары и часть капитальных благ;

- Строительство, $N_{4} = 25$. Отрасль создает новые производственные мощности и наполняет матрицу $B(t)$;

- Сельское хозяйство, $N_{5} = 34$. Отрасль обеспечивает значимую часть конечного спроса и использует ресурсы других отраслей.

Число фирм в каждой отрасли варьируется в пределах от 25 до 50 единиц. Такой диапазон выбран как компромисс между двумя требованиями. С одной стороны, он достаточен для отражения внутриотраслевой неоднородности фирм. С другой стороны, он позволяет сохранить вычислительную реализуемость модели на каждом шаге имитации. Различие числа агентов по секторам также содержательно оправдано: в обрабатывающей промышленности структура производителей более дробная, чем в строительстве, а в добывающем секторе и энергетике наблюдается относительно более высокая концентрация, хотя неоднородность фирм по масштабу и эффективности также остается существенной.

Пусть множество фирм в отрасли $j$ задается как

$$\mathcal{F}_{j} = \left\{ 1,2,\ldots,N_{j} \right\}.\quad\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ (4.2)$$

Тогда совокупный выпуск отрасли определяется суммой выпусков фирм:

$$x_{j}(t) = \sum_{f \in \mathcal{F}_{j}}^{}x_{jf}(t),\quad\quad j \in J.\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.3)$$

Тем самым отрасль в модели уже не выступает как однородный производственный блок, а рассматривается как результат взаимодействия множества агентов. Такая постановка соответствует логике агентного моделирования, в рамках которой макроэкономические зависимости выводятся из микроуровневых решений \[10\].

## 4.2 Основные предпосылки и временная структура

Модель строится в дискретном времени:

$$t = 0,1,2,\ldots,T.\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.4)$$

Один шаг модели интерпретируется как годовой или квартальный интервал, в зависимости от выбранной базы статистических данных.

Далее принимаются следующие предпосылки:

- Цены $p_{j}(t)$ считаются экзогенными или нормированными. Это позволяет сосредоточиться на материально-вещественной и инвестиционной структуре системы, не вводя отдельный ценовой блок;

- Каждая фирма характеризуется ограниченной рациональностью: она не решает глобальную оптимизационную задачу, а реагирует на наблюдаемый спрос, прибыль, дефицит мощностей и собственное технологическое состояние;

- Выпуск фирмы ограничен как ожидаемым спросом, так и производственной мощностью, определяемой капиталом и эффективностью его использования;

- Инвестиции имеют двойственную природу: часть инвестиций определяется прибылью, а часть -- необходимостью расширения мощностей при превышении спроса над текущим выпуском;

- Агрегированные коэффициенты прямых затрат и капиталоемкости пересчитываются на каждом шаге моделирования и потому являются эндогенными переменными модели.

## 4.3 Макроуровень. Динамический межотраслевой баланс

Базовым макроэкономическим каркасом модели выступает динамический межотраслевой баланс Леонтьева. В дискретной постановке его удобно записать в форме

$$x(t) = A(t)x(t) + B(t)\left( x(t + 1) - x(t) \right) + u(t),\quad\ \ \ \ \ \ \ \ \ \quad(4.5)$$

где $x(t) = \left( x_{1}(t),\ldots,x_{5}(t) \right)^{\top}$ - вектор валовых выпусков отраслей,

$A(t) = \left( a_{ij}(t) \right)$ - матрица прямых затрат,

$B(t) = \left( b_{ij}(t) \right)$ - матрица капиталоемкости,

$u(t) = \left( u_{1}(t),\ldots,u_{5}(t) \right)^{\top}$ - вектор конечного спроса \[8; 11; 14\].

Экономический смысл уравнения (4.5) состоит в следующем. Валовый выпуск каждой отрасли должен покрывать текущие промежуточные затраты, инвестиционные затраты, связанные с приростом выпуска, а также конечный спрос. В классической модели коэффициенты $A$ и $B$ задаются извне. В предлагаемой модели эти матрицы далее будут определяться через поведение фирм.

Если матрица $B(t)$ невырождена, из (4.5) можно получить явное выражение для целевого вектора выпуска следующего периода:

$$x(t + 1) = x(t) + B^{- 1}(t)\left( \left( I - A(t) \right)x(t) - u(t) \right).\quad\ \ \ \ \ \ \quad(4.6)$$

Если же $B(t)$ вырождена или близка к вырожденной, что содержательно возможно для отдельных конфигураций межотраслевого капиталообразования, вместо обратной матрицы используется псевдообратная матрица Мура - Пенроуза \[5\]:

$$x(t + 1) = x(t) + B^{+}(t)\left( \left( I - A(t) \right)x(t) - u(t) \right).\quad\ \ \ \ \ \ \ \quad(4.7)$$

Вектор $x(t + 1)$ далее интерпретируется как целевой отраслевой выпуск, который должен быть обеспечен микроуровнем. Таким образом, макроуровень задает системе нормативный баланс, а микроуровень определяет, насколько этот баланс может быть фактически реализован имеющимся набором фирм и их производственных возможностей.

Для пятиотраслевой структуры целесообразно сразу задать несколько содержательных ограничений на ненулевые элементы матриц $A(t)$ и $B(t)$. Во-первых, электроэнергетика поставляет универсальный ресурс всем отраслям, поэтому

$$a_{2j}(t) > 0,\quad\quad j = 1,\ldots,5.\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad\quad(4.8)$$

Добывающая промышленность обеспечивает топливно-сырьевую базу для энергетики, обработки, строительства и сельского хозяйства, следовательно,

$$a_{1j}(t) > 0,\quad\quad j = 2,3,4,5.\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.9)$$

Строительство является главным поставщиком новых производственных мощностей, поэтому соответствующие элементы матрицы капиталоемкости должны быть положительными:

$$b_{4j}(t) > 0,\quad\quad j = 1,\ldots,5.\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.10)$$

Обрабатывающая промышленность участвует как в промежуточном обеспечении производства, так и в формировании инвестиционных товаров, поэтому для нее естественно ожидать одновременно ненулевые элементы как в строке $a_{3j}(t)$, так и в строке $b_{3j}(t)$ при ряде $j$.

Вектор конечного спроса также имеет отраслевую специфику. Для сельского хозяйства и части продукции обрабатывающей промышленности доля конечного спроса выше, тогда как для добывающей промышленности и электроэнергетики доминирует промежуточное потребление. Тогда:

$$v_{j}(t) = \frac{u_{j}(t)}{x_{j}(t)},\quad\quad j \in J,\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.11)$$

причем в среднем ожидается

$$\nu_{5}(t) > \nu_{1}(t),\quad\quad\nu_{5}(t) > \nu_{2}(t).\quad\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ (4.12)$$

Соотношение (4.12) не является жестким ограничением, но служит содержательной проверкой реалистичности параметризации модели.

## 4.4 Микроуровень. Фирма как агент

Каждая фирма $f$ в отрасли $j$ описывается вектором состояния

$$s_{jf}(t) = \left( k_{jf}(t),\ \theta_{jf}(t),\ d_{jf}^{e}(t),\ \omega_{jf}(t),\ \pi_{jf}(t) \right),\quad\quad\ \ \ \ \ \ \ \ (4.13)$$

где $k_{jf}(t)$ -- запас производственного капитала,

$\theta_{jf}(t)$ -- технологический уровень фирмы,

$d_{jf}^{e}(t)$ -- ожидаемый спрос на продукцию фирмы,

$\omega_{jf}(t)$ -- ее рыночная доля внутри отрасли,

$\pi_{jf}(t)$ -- прибыль.

Начальная неоднородность фирм задается распределениями параметров. Для капитала удобно принять распределение,

$$k_{jf}(0) \sim LogN\left( \mu_{j},\sigma_{j}^{2} \right),\quad\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ (4.14)$$

что позволяет описать существование небольшого числа крупных компаний и большего числа средних и малых фирм. Начальный технологический уровень можно задать на отрезке

$$\theta_{jf}(0) \sim U\left\lbrack {\underline{\theta}}_{j},{\overline{\theta}}_{j} \right\rbrack,\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.15)$$

где границы интервала отражают отраслевую неоднородность технологий. Для добывающей промышленности и электроэнергетики этот интервал может быть уже, чем для обработки и сельского хозяйства, где различия между фирмами обычно выше.

Рыночные доли на начальном шаге удовлетворяют условию нормировки

$$\sum_{f \in \mathcal{F}_{j}}^{}\omega_{jf}(0) = 1,\quad\quad\omega_{jf}(0) > 0.\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.16)$$

В рамках модели фирма не максимизирует межвременную полезность или дисконтированную прибыль на всем горизонте. Вместо этого она действует по адаптивным правилам, отражающим ограниченную рациональность. Поэтому эволюцию ее состояния можно описать оператором

$$s_{jf}(t + 1) = \Phi_{jf}\left( s_{jf}(t),\Omega(t) \right),\quad\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ (4.17)$$

где $\Omega(t)$ -- вектор макроэкономической среды: отраслевой выпуск, целевой спрос, цены, параметры межотраслевых связей и правила регулирования.

## 4.5 Уравнения фирм

Целевой отраслевой выпуск, определенный на макроуровне, распределяется между фирмами пропорционально текущим рыночным долям:

$$d_{jf}(t + 1) = \omega_{jf}(t)\, x_{j}^{*}(t + 1).\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.18)$$

Однако фирма принимает решение не по самому отраслевому целевому спросу, а по его ожидаемому значению. Для этого используется адаптивное правило:

$$d_{jf}^{e}(t + 1) = \alpha_{j}d_{jf}^{e}(t) + \left( 1 - \alpha_{j} \right)d_{jf}(t + 1),\quad\quad 0 < \alpha_{j} < 1.\quad\ \ \quad(4.19)$$

Чем больше параметр $\alpha_{j}$, тем выше инерционность ожиданий. Для более волатильных отраслей, например для сельского хозяйства или добывающей промышленности, можно использовать меньшее значение $\alpha_{j}$, а для энергетики и строительства - большее.

Производственная мощность фирмы связывается с капиталом и технологическим уровнем. В линейном приближении мощность фирмы задается выражением

$${\bar{x}}_{jf}(t + 1) = \nu_{j}\,\theta_{jf}(t)\, k_{jf}(t),\quad\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ (4.20)$$

где $\nu_{j}$ -- коэффициент капиталоотдачи в отрасли $j$.

Тогда фактический выпуск фирмы определяется минимумом из ожидаемого спроса, выделенного ей целевого спроса и производственной мощности:

$$x_{jf}(t + 1) = min\left\{ d_{jf}^{e}(t + 1),\ d_{jf}(t + 1),\ {\bar{x}}_{jf}(t + 1) \right\}.\quad\ \ \ \ \ \ \quad(4.21)$$

Такая форма уравнения отражает одновременно спросовое и мощностное ограничение. Она особенно удобна для агентной симуляции, поскольку не требует решения сложной оптимизационной задачи на уровне отдельной фирмы.

Пусть $a_{ij}^{(f)}(t)$ -- индивидуальный технологический коэффициент фирмы $f$ отрасли $j$, показывающий, сколько продукции отрасли $i$ требуется для производства единицы выпуска этой фирмы. Тогда промежуточный спрос фирмы определяется как

$$z_{ijf}(t + 1) = a_{ij}^{(f)}(t)\, x_{jf}(t + 1),\quad\quad i,j \in J.\quad\ \ \ \ \ \ \ \ \ \ \ \quad(4.22)$$

Именно через уравнение (4.22) происходит микроэкономическое формирование будущей матрицы прямых затрат. Если фирма внедряет ресурсосберегающую технологию, то ее индивидуальные коэффициенты $a_{ij}^{(f)}(t)$ снижаются, и после агрегирования это приводит к изменению соответствующего столбца матрицы $A(t)$.

В условиях фиксированных или нормированных цен прибыль фирмы может быть задана как

$$\pi_{jf}(t + 1) = p_{j}(t)\, x_{jf}(t + 1) - \sum_{i = 1}^{5}p_{i}(t)\, z_{ijf}(t + 1) - c_{jf},\quad\ \ \ \ \ \ \ \ \quad(4.23)$$

где $c_{jf}$ -- условно-постоянные издержки.

Уравнение (4.23) позволяет связать текущий производственный выбор фирмы с ее будущей инвестиционной активностью. В частности, если прибыль оказывается отрицательной, фирма не имеет внутренних ресурсов для расширения производства, а значит ее инвестиционный спрос должен сокращаться.

Инвестиции фирмы состоят из двух компонент -- части текущей прибыли и части, обусловленной дефицитом мощности. Поэтому вводится следующее уравнение:

$$I_{jf}(t + 1) = \kappa_{j}\max\left\{ \pi_{jf}(t + 1),0 \right\} + \eta_{j}\max\left\{ d_{jf}(t + 1) - x_{jf}(t + 1),0 \right\},\quad\quad(4.24)$$

где $\kappa_{j}$ -- доля прибыли, направляемая на инвестиции,

$\eta_{j}$ - коэффициент реакции на неудовлетворенный спрос.

Для капиталоемких отраслей, например энергетики и добычи, параметр $\eta_{j}$ может быть ниже, чем для обработки или сельского хозяйства, так как наращивание мощности там медленнее и требует более крупных проектов.

Покупка инвестиционных благ по отраслям задается матрицей отраслевых долей:

$$q_{ijf}(t + 1) = \gamma_{ij}^{(f)}\, I_{jf}(t + 1),\quad\quad\sum_{i = 1}^{5}\gamma_{ij}^{(f)} = 1.\quad\quad\ \ \ \ \ \ \ \ \ \ \ \ (4.25)$$

Коэффициенты $\gamma_{ij}^{(f)}$ отражают структуру инвестиционного спроса фирмы. Для большинства отраслей наиболее значимой будет компонента $\gamma_{4j}^{(f)}$, связанная со строительством. Для обработки и энергетики существенной остается и доля капитальных товаров из самой обрабатывающей промышленности.

Запас капитала обновляется по стандартному закону накопления:

$$k_{jf}(t + 1) = \left( 1 - \delta_{j} \right)k_{jf}(t) + I_{jf}(t + 1),\quad\quad 0 < \delta_{j} < 1,\quad\ \ \ \ \ \quad(4.26)$$

где $\delta_{j}$ -- норма амортизации в отрасли $j$.

Более высокий износ обычно характерен для транспортабельного оборудования и машин, тогда как для части инфраструктурных фондов темпы амортизации ниже.

Технологический уровень фирмы повышается за счет части инвестиций, направленных на обновление оборудования и организационные изменения:

$$\theta_{jf}(t + 1) = \theta_{jf}(t)\left( 1 + \mu_{j} \bullet \frac{I_{jf}(t + 1)}{k_{if}(t)} \right),\quad\quad\mu_{j} > 0.\quad\quad\ \ \ \ \ \ \ \ \ (4.27)$$

Благодаря уравнению (4.27) технологический прогресс становится эндогенным результатом инвестиционной активности агентов. Это позволяет объяснить изменение коэффициентов прямых затрат через процессы внутриотраслевого отбора и обновления.

Рыночная доля фирмы на следующем шаге может определяться через долю фактического выпуска в совокупном выпуске отрасли:

$$\omega_{jf}(t + 1) = \frac{x_{jf}(t + 1)}{\sum_{g \in \mathcal{F}_{j}}^{}x_{jg}(t + 1)}.\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.28)$$

Если фирма систематически не удовлетворяет выделенный ей спрос или имеет низкий технологический уровень, ее рыночная доля со временем сокращается. В противоположном случае более эффективные производители усиливают свои позиции.

## 4.6 Переход от микроуровня к макроуровню

Агрегирование индивидуальных решений фирм образует общеотраслевые показатели и матрицы межотраслевого баланса.

Валовый выпуск отрасли в период $t + 1$ вычисляется как

$$x_{j}(t + 1) = \sum_{f \in \mathcal{F}_{j}}^{}x_{jf}(t + 1).\quad\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ (4.29)$$

Коэффициенты прямых затрат определяются как отношение совокупного использования промежуточного ресурса к совокупному выпуску отрасли:

$$a_{ij}(t + 1) = \frac{\sum_{f \in \mathcal{F}_{j}}^{}z_{ijf}(t + 1)}{\sum_{f \in \mathcal{F}_{j}}^{}x_{jf}(t + 1)}.\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.30)$$

Коэффициенты капиталоемкости формируются как отношение совокупного инвестиционного спроса на продукцию отрасли $i$ к приросту выпуска отрасли $j$:

$$b_{ij}(t + 1) = \frac{\sum_{f \in \mathcal{F}_{j}}^{}q_{ijf}(t + 1)}{max\{ x_{j}(t + 1) - x_{j}(t),\varepsilon\}},\quad\quad\varepsilon > 0.\quad\ \ \ \ \quad(4.31)$$

Параметр $\varepsilon$ вводится для предотвращения деления на нуль в случаях, когда прирост выпуска близок к нулю или отрицателен. В практической реализации величина $\varepsilon$ выбирается малой по сравнению со средним отраслевым выпуском.

Вектор конечного спроса можно разложить на три компоненты:

$$u_{j}(t) = u_{j}^{h}(t) + u_{j}^{g}(t) + u_{j}^{e}(t),\quad\quad j \in J,\quad\quad\ \ \ \ \ \ \ \ \ (4.32)$$

где $u_{j}^{h}(t)$ -- спрос домашних хозяйств,

$u_{j}^{g}(t)$ -- государственный спрос,

$u_{j}^{e}(t)$ -- внешний спрос.

Для сельского хозяйства обычно преобладает домашнее потребление, для строительства - государственный и инвестиционный спрос, а для добывающей промышленности существенную роль может играть внешний спрос.

После агрегирования микропоказателей формируются новые матрицы $A(t + 1)$ и $B(t + 1)$, а затем на их основе вычисляется целевой выпуск следующего периода по формулам (4.6) или (4.7). Тем самым система замыкается:

$$\left( A(t),B(t),u(t),x(t) \right) \rightarrow x^{*}(t + 1) \rightarrow \{ x_{jf}(t + 1),I_{jf}(t + 1),k_{jf}(t + 1)\} \rightarrow \left( A(t + 1),B(t + 1),x(t + 1) \right).\quad\quad(4.33)$$

Формула (4.33) выражает главный принцип построенной модели: макроуровень задает балансовый ориентир, а микроуровень наполняет его конкретным экономическим содержанием.

## 4.7 Условия корректности и допустимости модели

Для того чтобы агентная симуляция оставалась согласованной с динамическим межотраслевым балансом, необходимо контролировать ряд математических ограничений.

Прежде всего должна выполняться неотрицательность основных переменных:

$$x_{j}(t) \geq 0,\quad x_{jf}(t) \geq 0,\quad I_{jf}(t) \geq 0,\quad k_{jf}(t) \geq 0,\quad u_{j}(t) \geq 0.\quad(4.34)$$

Далее матрица прямых затрат должна оставаться продуктивной. В прикладной форме это требование удобно проверять через спектральный критерий \[8; 11; 12\]:

$$\rho\left( A(t) \right) < 1.\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.35)$$

Если условие (4.35) нарушается, значит совокупные технологические коэффициенты стали чрезмерно ресурсоемкими, и соответствующая конфигурация агентов не может рассматриваться как экономически допустимая.

Для матрицы капиталоемкости существенным является условие корректной обработки возможной вырожденности. Если

$$\det B(t) = 0,\quad\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad(4.36)$$

то при вычислении целевого выпуска должен использоваться не обычный оператор обращения, а псевдообратная матрица $B^{+}(t)$ согласно формуле (4.7) \[5\]. Это особенно важно для конфигураций, в которых часть отраслей почти не производит капитальных благ.

Наконец, необходимо контролировать остаток межотраслевого баланса. Пусть

$$r(t + 1) = x(t + 1) - A(t + 1)x(t + 1) - B(t + 1)\left( x(t + 2) - x(t + 1) \right) - u(t + 1).\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \quad\quad(4.37)$$

Вектор $r(t + 1)$ интерпретируется как показатель рассогласования между фактической агентной траекторией и балансовым нормативом. В идеальном случае он должен стремиться к нулю. На практике допускается малая ненулевая невязка, связанная с дискретизацией, округлениями и ограничениями мощности отдельных фирм.
