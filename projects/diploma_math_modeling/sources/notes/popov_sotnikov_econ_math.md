## Тезис работы (1-2 предложения)

Учебник по экономико-математическим методам и моделям (базовый курс, 2-е издание). Включает полное изложение модели МОБ Леонтьева, линейной модели обмена и моделей экономического роста (Солоу, Маркса-Моисеева).

---

## Ключевые аргументы

1. **Глава 10, §10.1** (с. 255–258) — «Модели межотраслевого баланса. Модель Леонтьева»: обозначения xᵢⱼ, Xᵢ, Yᵢ; коэффициенты прямых затрат aᵢⱼ = xᵢⱼ/Xⱼ; балансовое уравнение X = AX + Y (уравнение 10.3); решение X = (E−A)⁻¹Y (уравнение 10.4). Матрица (E−A)⁻¹ — матрица полных затрат.
2. **§10.2** (с. 258–260) — Линейная модель обмена: структурная матрица торговли, равновесие AX = X (уравнение 10.7).
3. **§10.3** (с. 260–264) — Модели развития экономики: упрощённая модель с инвестициями I(t) = B·dY/dt (коэффициент капиталоёмкости B) → дифференциальное уравнение роста дохода Y(t); модель Солоу (производственная функция F(K,L), выбытие капитала δ, условие равновесия af(k*) = (δ+n+g)k*, «золотое правило» f'(k**) = δ+n+g).

## Методология

Учебник. Алгебраические методы МОБ. Теория дифференциальных уравнений (непрерывное время). Модель Солоу — производственная функция + оптимальная норма инвестиций.

## Релевантные цитаты

> "Модель [МОБ] была разработана в 1936 г. американским экономистом [Леонтьевым]. Введём понятие коэффициентов прямых затрат aᵢⱼ = xᵢⱼ/Xⱼ... Систему уравнений... можно записать в матричном виде X = AX + Y." (с. 255–256)

> "Коэффициент пропорциональности B называется коэффициентом капиталоёмкости прироста дохода. I(t) = B·dY/dt." (с. 259, упрощённая модель роста)

---

## Связь с моей работой

- Учебный источник для формального введения МОБ в Главе 1 диплома.
- Параметр B в §10.3 как «коэффициент капиталоёмкости прироста дохода» — перекликается с матрицей B(t) в дипломе (акселератор), но здесь B — скаляр для агрегированной модели.
- Модель Солоу (§10.3) — контекст для анализа устойчивого роста, полезна для сравнения с дипломной моделью.
