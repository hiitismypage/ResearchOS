## Исследование

- **Исследовательский вопрос:** Как двухуровневая гибридная модель (динамический МОБ Леонтьева + агентно-ориентированный микроуровень) позволяет воспроизвести макроэкономическую динамику и оценить устойчивость системы к структурным шокам?

- **Гипотеза:** Интеграция межотраслевого баланса с гетерогенными агентами, принимающими решения по правилам ограниченной рациональности, обеспечивает более реалистичное описание переходных процессов по сравнению с моделями с репрезентативным агентом.

- **Цель работы:** Разработать и верифицировать двухуровневую динамическую модель макроэкономики, откалиброванную по реальным данным, и исследовать её поведение при структурных шоках.

- **Задачи работы:**
  1. Построить динамическую матричную модель МОБ с эндогенными матрицами A(t) и B(t) для 5-секторной экономики
  2. Разработать микроуровень: 159 гетерогенных фирм-агентов с правилами ограниченной рациональности
  3. Откалибровать модель по данным Eurostat (naio_10_cp1700, 2015) и временным рядам 2010–2019
  4. Реализовать модель на Python и верифицировать на исторических данных
  5. Провести сценарный анализ: ответ системы на структурные шоки

- **Методология:**
  - Макроуровень: динамический МОБ Леонтьева, матричные дифференциальные уравнения
  - Микроуровень: агентно-ориентированное моделирование (ABM), ограниченная рациональность (Simon)
  - Калибровка: данные Eurostat (Германия, не указывается явно в тексте) + временные ряды 2010–2019
  - Реализация: Python (numpy, scipy, pandas, python-docx)

- **Объект исследования:** Макроэкономическая система государства как динамическая межотраслевая структура

- **Предмет исследования:** Механизмы взаимодействия отраслевых агентов и их влияние на макроэкономическую динамику
